Project Title

Interactive Map of Massachusetts

Description:

I would like to create an interactive map to some experience displaying a yet to be determined data set (likely something like 311 information, temperature, landmarks, crime from a spreadsheet/xml file which depends upon what is readily available) on a fixed map of Massachusetts.

Emphasis will be placed on experimenting with different types of interactive. I will also make use of existing JQuery map plugins. If all goes well I may use this as the foundation for P4

Sketch

Am not sure what is possible without further investigation, but layout will likely be a full map of Massachusetts with ability pan/zoom. Map key which will enable user to toggle on and off visibility of items on map and/or show the same information for a certain duration of time.
